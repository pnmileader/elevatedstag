# Design System — The Elevated Stag CRM

## Design Context

- **Target audience**: Katie Fore, luxury men's custom clothier. Power user who manages 35+ clients daily. Checks CRM on laptop between appointments, occasionally on phone.
- **Use cases**: Track client pipeline, monitor orders in progress, manage follow-ups and deadlines, review revenue, schedule fittings.
- **Brand personality**: Sophisticated masculine luxury. Sharp, confident, warm. Like a well-tailored suit — precise but not cold.

## Aesthetic Direction

**Bloomberg-dense meets luxury bespoke.** Information-packed like a trading terminal, but with the warmth and refinement of a high-end menswear atelier. Every pixel earns its place.

## Density Rules

- **Base font**: 13px (0.8125rem)
- **Spacing grid**: 4px increments (4, 8, 12, 16, 20, 24, 32)
- **Row height**: 32px for table rows, 36px for interactive rows
- **Section gaps**: 16px between sibling sections, 24px between major zones
- **Card padding**: 16px (compact), 20px (standard)
- **No decorative whitespace** — hierarchy through weight and color, not space

## Typography

- **Headings**: Cormorant Garamond, 600 weight. Used for page titles and section headings only.
- **Body**: DM Sans, 400/500/600 weight. Everything else.
- **Scale**: 11px labels → 13px body → 15px subheadings → 20px page titles → 28px hero metrics
- **Letter-spacing**: 0.08em on uppercase labels, normal everywhere else

## Components

### Sidebar
- Collapsed (icon-only, 48px) by default on desktop
- Expandable on hover or toggle
- Dark charcoal background
- Active item: cream text + gold left border accent (2px)
- Inactive: cream/50 opacity

### Tables
- 32px row height
- Alternating row tint (warm-white / ivory)
- Header row: uppercase 11px labels, gray-dark color, bottom border
- Hover: ivory background
- No vertical borders, horizontal borders only (gray-med)

### Cards (when used)
- 16-20px padding
- 1px gray-med border, no rounded corners
- No box-shadow by default; shadow-sm on hover
- Alternating tint between adjacent cards

### Buttons
- Primary: charcoal bg, cream text, no rounding, 32px height
- Secondary: 1px gray-med border, body text, no rounding
- Ghost: text-only, gold color, uppercase 11px

### Status Badges
- No rounding, no background fills
- Uppercase 10px text with colored left border (2px)
- Color-coded: ordered=gray, cutting=blue, sewing=amber, shipping=teal, delivered=green

### Form Inputs
- 32px height, 12px horizontal padding
- 1px gray-med border, no rounding
- Gold border on focus
- 13px font size

## Layout Rules

- Sidebar + main content flex layout
- Main content: overflow-x-hidden, overflow-y-auto
- Max content width: 1100px
- Page padding: 24px on desktop, 16px on mobile
- Grid gaps: 12-16px between items
- Section separators: 1px gray-med border-top with 16px vertical margin
